    So, what is Scala at a high level?

    “Scala is an object-functional programming and scripting language for general software applications.”

    “Scalable + Language = Scala, the language that grows with demands of users.”

    - Wikipedia

    A Few High Level Features

    • Object and Functional Language
    • Strongly Typed
    • Built in Concurrency Patterns
    • Pattern Matching
  • Type system is compile time checked
  • Encouraged to treat as a dynamic language to let/learn how compiler works
  • Immutability is encourages in variables and messages
  • Language is split up in Scala and Akka
  • Think regex in the class and method declarations and processing

    • Some Basics

    Trait and Case Class

    
trait Bike {
    val brand: String
    val radius: Double

    def calculateCircumference(): Double = 2 * Pi * radius
}

case class Serrota(brand:String="Serrota", radius:Integer=311) extends Bike

case class TrekMtnBike(brand:String="Trek", radius:Integer=292) extends Bike

    Pattern Matching

    
object MatchTest1 extends Application {
    def matchTest(x: Int): String = x match {
        case 1 => "one"
        case 2 => "two"
        case _ => "many"
    }

    println(matchTest(3))
}

    Why...

  • Scala offers powerful features for abstraction that differ from Java
  • Pooling and Phases are abstracted into Akka framework
  • Runs on the JVM:
    • Use existing Java modules
    • Maven and Spring
    • Use success of Java language

    • Take aways...

    • Concurrency built with Akka Framework and abstracted from application
    • Actor Pattern is powerful with immutable messages
    • Language speed is based on JVM and Java NIO
    • Functional nature is well suited for concurrency problems
